GNU Queue

Introduction

GNU Queue is a program to spawn tasks on a pool of separate computer systems. GNU Queue load balances across the resources available in a compute pool. A user submits a task and the first available system meeting the resources requirements will execute the task. The current design direction is to assume essentially identical independent systems.
